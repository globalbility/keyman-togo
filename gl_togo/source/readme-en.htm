A keyboard layout for the languages of Togo.

This keyboard layout is designed for the languages of Togo, and is based on the French keyboard layout.

The following languages are supported: Aja, Anii, Ewe, Fon, Fulfulde, Gen, Ife, Ikposo, Kabiye, Lama, Lukpa, Miyobe, Moba, Nawdm, Ngangam, Ntcham, Tem.
